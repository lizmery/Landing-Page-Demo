# Landing Page Demo
A website demo featuring a landing page with animations.
[View Live](https://landingpage-example.netlify.app/)

## Technologies Used
- CSS3
- HTML5
- JavaScript

## Features
- CSS animations using keyframes
- Responsive design
- Portfolio section
- Contact form
- Accordion for FAQs (JavaScript)
